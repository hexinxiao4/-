# 电子物证理论考试刷题库

> 📚 一个基于纯静态网页的电子物证专业知识刷题系统，支持单选、多选、判断三种题型，内置 **520 道**精选题目，适用于电子物证检验鉴定职称考试、公安机关电子数据取证专业考试等备考复习。

[![题型](https://img.shields.io/badge/题型-单选%2F多选%2F判断-green)](https://github.com/)
[![部署](https://img.shields.io/badge/部署-GitHub%20Pages-orange)](https://github.com/)

---

## ✨ 功能特性

- **520 道精选题目** — 覆盖电子物证考试核心知识点
- **三种题型** — 单选题（230题）、多选题（90题）、判断题（200题）
- **随机练习** — 按题型分门别类，打乱顺序刷题
- **模拟考试** — 40题限时模拟（60分钟），检验真实水平
- **错题本** — 自动记录错题，针对性复习
- **收藏夹** — 收藏重点题目，随时回顾
- **学习进度** — 自动保存答题记录，刷新不丢失
- **详细解析** — 每题附参考答案与详细解析
- **响应式设计** — 支持电脑、平板、手机多端访问

## 🚀 在线使用

### 方式一：直接打开（本地）
```bash
# 下载后直接用浏览器打开 index.html 即可
```

### 方式二：部署到 GitHub Pages（推荐）

1. **Fork 本仓库** 或创建新仓库，将 `index.html` 和 `questions.js` 上传
2. 进入仓库 **Settings → Pages**
3. 在 **Source** 中选择 **Deploy from a branch**
4. 分支选择 `main` 或 `master`，文件夹选择 `/(root)`
5. 点击 **Save**，等待约 1 分钟
6. 访问 `https://你的用户名.github.io/仓库名/` 即可在线刷题

> 💡 所有答题记录通过浏览器 `localStorage` 保存在本地，更换设备或清除浏览器数据会丢失进度。

## 🛠️ 技术栈

- **纯前端实现**：HTML5 + CSS3 + Vanilla JavaScript
- **无后端依赖**：无需服务器，无需数据库
- **本地存储**：`localStorage` 持久化答题进度
- **响应式布局**：CSS Grid + Flexbox，适配多端

## 📖 使用技巧

| 功能 | 操作 |
|------|------|
| 切换题型 | 点击左侧侧边栏「单选题 / 多选题 / 判断题」 |
| 查看答案 | 作答后点击「提交答案」或直接查看 |
| 收藏题目 | 点击题目右上方「⭐ 收藏」按钮 |
| 模拟考试 | 点击「模拟考试」进入 40 题限时考试 |
| 查看错题 | 点击「错题本」复习所有答错的题目 |
| 清空进度 | 在错题本页面点击「清空错题本」 |
